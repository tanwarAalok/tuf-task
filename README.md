## How to run locally 
- Clone the repository
 ```
  git clone https://github.com/tanwarAalok/tuf-task.git
  cd tuf-task
  ```
- Start backend
  - First create .env file and add
    ```
      DB_HOST=
      DB_USER=
      DB_PASSWORD=
      DB_NAME=
    ```
  - Then in terminal
    ```
    npm i
    npm start
    ```
- Start Frontend
  - First change the API_URL path in src/apiFetchers.js file
  - Then in terminal
    ```
    npm i
    npm start
    ```
